Middleware Solutions

Discord Notifications

Middleware postCheckin onError
Author: mastersamasama
Last updated: 2024-03-18

Real-time notification system for check-in results through Discord webhooks. Supports multi-language notifications and smart mention features.

Configuration Guide

Basic Configuration

{
    name: "discord-notify",
    modulePath: "@official/discord-notify.js",
    target: ["*"],
    options: {
        webhook: "YOUR_WEBHOOK_URL",
        language: "zh-tw",
        tag_filter: [0],
        mentionUsers: ["USER_ID"]
    }
}
Security Note: Always store webhook URLs in environment variables

Configuration Options

Option Description Default
webhook Discord webhook URL (required) -
language Notification language, support 'en', 'zh-cn', 'zh-tw', 'ja', 'ko' "en"
tag_filter Status codes to mute notifications for [0]
mentionUsers User IDs to mention on failure, @ that user []

Status Code Reference

Success: 0
Already Checked In: -5003
Cookie Expired: -100

Cookies Formatter

Middleware preCheckin
Author: mastersamasama
Last updated: 2024-03-18

Automatically processes and sanitizes cookies from browser copy-paste format. Ensures proper cookie formatting before check-in execution.

Configuration Guide

Basic Configuration

{
    name: "cookies-tidy",
    modulePath: "@official/cookies-tidy.js",
    target: ["*"]
}
Automatic Processing: No options required - works automatically on raw cookie strings

Transformation Example

Input: "ltoken=abc123; ltuid=123456; "
Output: "ltoken=abc123;ltuid=123456"

Common Issues Handled

  • Extra spaces between cookies
  • Trailing semicolons
  • Inconsistent formatting from browser copy
